# Return Largest Numbers in Arrays

Create a function that returns the largest number from each provided sub-array.

Example:

```javascript
largestOfFour([[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]); // => [5, 27, 39, 1001]
```
