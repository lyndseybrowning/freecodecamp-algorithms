# Falsy Bouncer

Create a function that removes all falsy values from an array.

Falsy values are: ```false```, ```null```, ```0```, ```''```, ```undefined``` and ```NaN```.

Example:

```javascript
bouncer([7, 'ate', '', false, 9]); // => [7, 'ate', 9]
```
