# Chunky Monkey

Create a function that splits an array into groups the length of size provided.

Return groups as a two-dimensional array.

Example:

```javascript
chunkyArrayInGroups(['a', 'b', 'c', 'd'], 2) // => [ ['a', 'b'], ['c', 'd']]
```
